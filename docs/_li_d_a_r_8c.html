Detailed Description

Implementation of the TF-Luna LiDAR UART driver.

The driver searches the incoming UART byte stream for the TF-Luna frame header, validates the checksum, decodes distance, strength, and temperature, and marks the measurement as valid only if it passes the configured range and signal-strength checks.
