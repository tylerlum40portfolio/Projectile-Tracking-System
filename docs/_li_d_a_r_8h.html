Detailed Description

TF-Luna LiDAR UART driver for projectile range measurement.

This driver receives and validates distance frames from the TF-Luna LiDAR sensor. The validated range measurement is used by the projectile tracking firmware to confirm whether the detected camera target is close enough for the laser to be enabled. The driver stores the latest distance, signal strength, temperature, and validity flag in a Lidar_t object.

enum LidarStatus_t

Status codes returned by the LiDAR driver.

These values allow the main application to distinguish between a successful read, a timeout, a bad checksum, a UART failure, or an invalid argument.

Enumerator
LIDAR_OK 

A valid frame was received and parsed.

LIDAR_TIMEOUT 

No complete frame was received before the timeout.

LIDAR_BAD_CHECKSUM 

A frame was received but failed checksum validation.

LIDAR_UART_ERROR 

HAL UART returned an error other than timeout.

LIDAR_NULL_ARG 

A NULL device pointer or UART handle was provided.
