Detailed Description

Pixy2 SPI target-detection driver implementation.

This file sends getBlocks requests to the Pixy2 camera, validates the response packet, decodes color-connected-component blocks, and selects the largest block matching the configured target signature. The selected block is converted into horizontal and vertical pixel error for the tracking controller.
