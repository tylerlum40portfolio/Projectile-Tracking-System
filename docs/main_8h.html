Detailed Description

: Common application header and CubeMX-generated pin map.

This file contains the STM32 HAL include, Error_Handler prototype, and GPIO pin definitions used by the projectile tracking firmware. Important signals include Pixy2 SPI chip select, laser control, pan/tilt motor direction pins, and motor-driver select pins.
