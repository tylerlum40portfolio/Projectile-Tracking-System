Detailed Description

Pan and tilt DC motor driver interface.

This driver abstracts the GPIO and PWM details used to command the projectile tracking mechanism. The pan motor is driven from TIM4 channel 3 and the tilt motor is driven from TIM4 channel 4. Direction is controlled with INA/INB GPIO pins and speed is controlled by PWM duty cycle.
