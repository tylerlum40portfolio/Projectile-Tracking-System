Detailed Description

: Main application for the LiDAR projectile tracking system.

This file integrates the Pixy2 camera, TF-Luna LiDAR sensor, laser module, and pan/tilt motor driver. The intended operating mode is a 20 Hz control loop triggered by TIM3. In each control cycle, the camera searches for the projectile target, PID control converts pixel error into pan/tilt motor commands, the LiDAR validates range, and the laser is enabled only when the target is centered and within the allowed distance.

◆ HAL_TIM_PeriodElapsedCallback()

void HAL_TIM_PeriodElapsedCallback ( TIM_HandleTypeDef * htim)

Timer period elapsed callback used by the 20 Hz control loop.

TIM3 triggers this callback at the configured control rate. The interrupt only sets a flag, while the main loop performs the actual camera, LiDAR, motor, and laser work. This avoids doing slow SPI/UART operations inside the interrupt.

Parameters
htimPointer to the timer handle that generated the interrupt.

int main ( void )

Application entry point.

Initializes the HAL, system clock, GPIO, PWM timer, SPI, UART, control timer, and custom project drivers. The main while loop currently contains a motor direction bench test, while the full camera/LiDAR/PID tracking loop is kept directly below it as the intended operating sequence.

Returns
This function should never return during normal embedded operation.
